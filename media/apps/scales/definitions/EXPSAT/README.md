# Explanation Satisfaction Scale

**Abbreviation:** EXPSAT  
**Code:** `EXPSAT`  
**Version:** 1.0  
**License:** CC  

Standard scale for evaluating user satisfaction in explanations provided by AI and XAI systems.

## Scale Summary

- **Items:** 0
- **Dimensions:** 1
  - **Overall satisfaction:** Mean satisfaction score

## Scoring

- **Overall**: mean_coded (7 items)

## Citation

> Hoffman, R. R., Mueller, S. T., Klein, G., & Litman, J. (2018). Metrics for explainable AI: Challenges and prospects. arXiv preprint arXiv:1812.04608.
>
> Hoffman, R. R., Mueller, S. T., Klein, G., & Litman, J. (2023). Measures for explainable AI: Explanation goodness, user satisfaction, mental models, curiosity, trust, and human-AI performance. Frontiers in Computer Science, 5, 1096257.

## Files

- `EXPSAT.osd` - Scale definition (OpenScales OSD format)

This scale is designed to be run using the PEBL ScaleRunner system.
