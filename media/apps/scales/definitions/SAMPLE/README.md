# Sample Scale with All Question Types

**Code:** `SAMPLE`  
**Version:** 1.0  

Demonstrates all 10 question types supported by ScaleRunner

## Scale Summary

- **Items:** 0
- **Dimensions:** 2
  - **Satisfaction**
  - **Engagement**

## Scoring

- **satisfaction**: sum_coded (2 items)
  - Sum of coded responses (q2 is reverse-coded)
- **engagement**: mean_coded (2 items)
  - Mean of coded responses
