Bei dieser Aufgabe handelt es sich um ein Glücksspiel. Sie spielen um Spielgeld. Ihr Startkapital beträgt 2000€. Bei jedem Versuch wählen Sie eine Karte von einem der vier Stapel.  Wenn Sie eine Karte ausgewählt haben bekommen Sie dafür eine Belohnung, manchmal müssen Sie jedoch gleichzeitig eine Geldstrafe zahlen. Ziel ist es den Gewinn zu maximieren. Sie können jederzeit von jedem Stapel eine Karte ziehen. Klicken Sie um fortzufahren.
Für jede Karte die Sie ziehen bekommen Sie eine Belohnung. Die Höhe dieser Belohnung hängt davon ab, welchen Stapel Sie wählen. Jeder Stapel hat eine fixe Belohnungshöhe. Sie werden aber auch Geldstrafen zahlen müssen. Diese Strafen sind einigermaßen zufällig und variieren von Stapel zu Stapel. Manchmal beträgt die Strafe 0€, manchmal ist sie sogar höher als die Belohnung die Sie für die Wahl einer Karte eines bestimmten Stapels bekommen haben. Probieren Sie den Gewinn zu maximieren. Klicken Sie um fortzufahren. 
Die Grafik am oberen Bildschirmrand zeigt Ihnen wie viel Spielgeld Sie besitzen. Der grüne Balken verlängert sich proportional zu ihrem Gewinn. Der rote Balken zeigt Ihnen Ihre Schulden von 2000€ an.
