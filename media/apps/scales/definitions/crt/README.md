# Cognitive Reflection Test

**Code:** `crt`  
**Version:** 1.0  

3-item test measuring the tendency to override an incorrect intuitive response and engage in further reflection

## Scale Summary

- **Items:** 3
- **Other items:** 3
- **Dimensions:** 1
  - **Cognitive Reflection**
- **Languages:** en

## Scoring

- **reflection**: sum_correct (3 items)
  - Number of correct responses (0-3)
