PEBL's Simon Interference Task Version 2.0. Judge the color of a stimulus while ignoring its horizontal position.

VERSION 2.0 UPDATES (2025):
- Migrated to Layout & Response System for unified response handling
- Added support for multiple response modes (keyboard, mouse, touch)
- Added JSON parameter file support with schema validation
- Updated translations for 8 languages to be response-mode-agnostic
- Added Continue button for mouse/touch modes during practice and breaks
- Improved visual layout with better positioning
- Modernized code for PEBL 2.2 compatibility

Data are saved to simon-X.csv and simon-summary.txt files in data/X/ directory.

To cite: Mueller, S. T. (2011). The PEBL Simon Interference Task. Computer software retrieved from http://pebl.sf.net/battery.html
