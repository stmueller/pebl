The PEBL Visual Search Task

Search for one or more targets defined by color or shape in a field of distractors.
Can be adapted into many visual search paradigms.

Duration: ~10-15 minutes
Response: Mouse clicks on target locations
Browsers: Chrome, Firefox, Safari, Edge (latest versions)

Parameters:
- search1size, search2size, search3size: Number of items in search array (10, 20, 30)
- targ1, targ2: Target letters (default: O, X)
- foil1-foil5: Distractor letters (default: U, D, G, C, Q)
- targcol1, targcol2: Target colors (default: white, green)
- foilcolor: Distractor color (default: white)
- numtargs1, numtargs2, numtargs3: Number of targets in each condition (0, 1, 5)
- reps: Number of repetitions per condition (default: 5)

Data Output:
- vsearch-SUBNUM.csv: Trial-by-trial data with search time and selection time
- summary-SUBNUM.txt: Condition averages (mean RT and accuracy by target/color/size)

Citation:
Mueller, S. T. (2011). The PEBL Visual Search Task. Computer software
retrieved from http://pebl.sf.net.
