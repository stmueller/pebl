# Cognitive Load Theory Questionnaire

**Abbreviation:** CLT  
**Code:** `clt`  
**Version:** 1.0  
**License:** Creative Commons Attribution 4.0 (CC BY 4.0)  

15-item scale measuring three types of cognitive load during learning

## Scale Summary

- **Items:** 15
- **Likert items:** 15 (9-point scale)
- **Dimensions:** 3
  - **Intrinsic Cognitive Load (ICL):** Cognitive load imposed by the inherent difficulty of the learning content
  - **Extraneous Cognitive Load (ECL):** Cognitive load imposed by the design of the learning material
  - **Germane Cognitive Load (GCL):** Cognitive load associated with actively processing and understanding the material
- **Languages:** de, en

## Scoring

- **ICL**: mean_coded (5 items)
  - Mean intrinsic cognitive load (1-9 scale)
- **ECL**: mean_coded (5 items)
  - Mean extraneous cognitive load (1-9 scale)
- **GCL**: mean_coded (5 items)
  - Mean germane cognitive load (1-9 scale)
