PEBL Function Reference¶

Welcome to the PEBL Function Reference. This documentation covers all functions available in the Psychology Experiment Building Language.

  • PEBLStream - File and Network I/O

  • PEBLString - String Manipulation

PEBL Library Functions:

  • Design Library - Experimental Design

  • Graphics Library - Advanced Graphics

  • HTML Library - HTML Generation

  • Math Library - Extended Mathematical

  • UI Library - User Interface

  • Utility Library - Helpers and Utilities
